Установка и настройка программы

Информация по обновлению ATSLog находится в файле UPDATING.

Требования

Для установки и работы ATSlog ваша система должна содержать следующие пакеты:
  • POSIX совместимая unix система. Проверялось и работает на Linux и FreeBSD. Скорее всего будет работать и на других UNIX OS.
  • Компилятор GCC
  • MySQL (проверялось с 4.xx и 5.xx) или PostgreSQL
  • Perl 5, модули Perl DBD-mysql или DBD-Pg modules для MySQL или PostgreSQL.
  • Если вы хотите использовать веб интерфейс - сервер http (проверялось с Apache HTTPD) с установленной поддержкой PHP (4.3 или выше).

Установка

Для FreeBSD пользователей существует порт atslog. Подробности на сайте www.freshports.org.
    1. Скачайте и распакуйте atslog-.tar.gz: 
      $ fetch 

$ tar vzxfmp atslog-.tar.gz
    2. Посмотрите опции файла configure. 
      $ ./configure --help
      После чего запустите configure скрипт без параметров или с вашими настройками. 
      $ ./configure
    3. Установите программу командой make install 
      # make install
    4. Отредактируйте конфигурационный файл программы, $PREFIX/atslog.conf (например, /usr/local/etc/atslog.conf). Обратите особое внимание на такие параметры как: $port, $speed, $fastwrite, $model. 
      # $EDITOR /usr/local/etc/atslog.conf
    5. Запустите ваш SQL сервер (PostgreSQL или MySQL). Создайте базу данный atslog командой 
      $ make install-sql
      Эта команда автоматически создаст базу данных atslog и все служебные таблицы.

    6. Для того, чтобы сделать доступным web интерфейс добавте его описание в httpd.conf, например так: 
          Alias /atslog/ "/usr/local/www/atslog/"
    <Directory "/usr/local/www/atslog/">
        Options none
        AllowOverride Limit
        Order Deny,Allow
        Deny from all
        Allow from 127.0.0.1 .example.com
    </Directory>
      Настройки веб интерфейса находятся файле www/include/config.inc.php. Скопируйте пароль для SQL из файла atslog.conf в этот файл.
    7. Внимательно прочитайте файл USAGE.RUS и запустите atslog командой atslogmaster start.
