Программы тарификации АТС — каталог программ тарификации.

ATC CLIENT-SERVER — бесплатный вариант биллинга под win32.

Комплекс "Статистика АТС" — для платформ win32.

Программа тарификации SMDR Tariff — бесплатна; win32.

TariScope (Atlas Billing) — биллинговый программный комплекс предназначен для учета телефонных переговоров, регистрируемых офисными или учрежденческими АТС. Коммерческий программный продукт.

ATS tarifficator — программа работает на платформе Win32. Качественный коммерческий программный продукт.

CDR Reader — данная программа предназначена для считывания информации о произошедших звонках с различных типов офисных АТС и записи их в текстовые файлы. Запись может производиться как в один единый, так и в отдельные файлы по дням или по месяцам при условии, что поддержка записей, выдаваемых конкретной PBX, есть в программе.

PBX Call Tarifficator — для платформы Win32.

PhoneTax — программа учета и тарификации телефонных звонков для АТС (Win32).

Phone Xpress — многообещающий коммерческий софт для Win32 платформ.

Serpand — Программа для занесения данных соединений телефонной станции Panasonic в базу данных MySQL. Для UNIX подобных платформ.
