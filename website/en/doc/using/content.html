Using ATSlog

/usr/local/etc/atslog.conf — config file.
atslogmaster — running without parameters shows daemon status and issues short help on atslogmaster program.
atslogmaster start — launching ATSlog.
atslogmaster stop — stopping ATSlog.
atslogmaster restart — restart.
atslogmaster writedb — writing last calls into the server SQL database.
atslogmaster rotate — log file rotation.
atslogmaster cleardb — cleaning up SQL server tables.
atslogmaster alltodb — writing all calls (including archive) into the server SQL database. Data on current calls from /var/log/atslog/curr entcalls.log are ignored because of possible risk of doubling entries. To wr ite the current calls as well use the function writedb
