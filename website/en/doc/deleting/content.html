Uninstalling
Uninstalling ATSlog. Step-by-step guide.
  1. Stop ATSlog:
    atslogmaster stop

  2. Change to the catalog indicated by the parameter $sharedir of the config file (by default /usr/local/share/atslog) and issue:
    make deinstall

    Upon deinstaling the script checks the files with working information in order not to delete something which it has not to

    Text log-files are not deleted (all contents of the catalog indicated by the par ameter $logdir in the config file; by default this is /var/log/atslog ) . The directories containing any files not belonging to ATSlog are kept.

    Data from SQL server tables are kept.

    Upon deinstalling the PID file ( /var/run/atslog.pid ) is kept.
