30 July 2005 Source code repository opening
  • ATSlog repository:
    • Since long ago the ATSlog software is being developed using the source code repository. It is simply in use for both upgrading and developing.
    • A SVN repository provided by the prohect berlios.de is used.
    • Getting SVN files from the repository:
      svn checkout svn://svn.berlios.de/atslog/trunk
    • The source tree needs final touch. It is simply: just copy the repository to some working directory and issue a command
      make version inside it.
    • At further editings there is no need to check out the whole repository from the very beginning. Just switch to the catalog and issue the command:
      svn update
