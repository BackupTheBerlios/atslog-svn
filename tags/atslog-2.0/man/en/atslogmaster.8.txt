NAME

ATSlog предоставляет удобный интерфейс с доступом через web-браузер для просмотра и анализа звонков различных моделей АТС

SYNOPSIS

atslogmaster [start] [stop] [restart] [writedb] [rotate] [cleardb] [alltodb]

DESCRIPTION

atslogmaster - управляет серверной частью ATSlog

OPTIONS

start			запуск ATSlog. 
stop			останов. 
restart			перезапуск. 
reload			synonym of "restart"
writedb			запись последних звонков в базу SQL сервера. 
rotate			ротация текстовых лог-файлов.
cleardb			очистка таблиц SQL сервера с информацией о звонках.
alltodb			запись всех (в т.ч. из архивов) звонков в базу SQL сервера. Данные о текущих звонках из файла			 /var/log/atslog/currentcalls.log не добавляются из-за возможного дублирования  информации. Если есть необходимость записи текущих звонков, то воспользуйтесь функцией writedb
			    

EXAMPLES

atslogmaster start - запуск демона для сбора данных.

SEE ALSO

atslogd(8), /usr/local/etc/atslog.conf
